# ⚠️ Watch Your Head

Watch Your Head é um mini game 2D no estilo endless runner, desenvolvido como um projeto pessoal usando a Godot.

É um dos jogos mais simples que fiz, com foco em acessibilidade e detalhes visuais.

## 🎮 Sobre o jogo

O objetivo é simples: **sobreviver o máximo de tempo possível**.

Serras circulares caem do céu constantemente, e o jogador deve se mover para os lados para desviar delas.

- 🎯 3 acertos = Game Over  
- ⏱️ Quanto mais tempo sobreviver, maior a pontuação  
- 🔁 Modo endless (sem fim)

## ⚙️ Mecânicas

Com o passar do tempo, o jogo vai ficando mais desafiador:

- Padrões diferentes de queda das serras  
- Aumento gradual da dificuldade  
- Situações mais imprevisíveis

## 🎨 Foco do projeto

Neste projeto, o foco principal foi trabalhar aspectos visuais e de feedback:

- Interface (UI)
- Efeitos simples com partículas
- Animações
- Elementos sonoros básicos

Apesar de simples na jogabilidade, foi um dos projetos onde mais explorei os detalhes visuais.

## 📚 Observação

Este projeto foi desenvolvido com foco em aprendizado e experimentação, então o código pode conter soluções mais simples ou não otimizadas.
